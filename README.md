# Super Calculadora
---
Esta aplicação é uma calculadora que tem como input 2 números onde calcula os valores de soma, fatorial, raiz quadrada, entre outros. Projeto desenvolvido no Bootcamp Frontend do IGTI 2020.

## Technologies

- Full JavaScript(ES6)
